Implementace

Projekt je vytvořen pomocí Python Web frameworku Django, kterým byla vytvořena i celá struktura adresářů a s jehož pomocí je vytvořena celá databáze (models.py). Formuláře pro přidávání záznamů do tabulek se nachází v souboru form.py. Funkcionalita stránek a propojení s databází se nachází v souboru views.py. Jednotlivé webové stránky jsou ve složce FestivalReservations/templates/FestivalReservations.

  • gunicorn 20.0.4
  • django 3.1.3
  • django-heroku 0.3.1
  • jsonfield 3.1.0
  • django-vanilla-views 2.0.0
  • Pillow 8.0.1
  • psycopg2-binary 2.8.6
  • dj-database-url 0.5.0
  • PostgreSQL 11+200+deb10u4
  • Pro stáhnutí všech potřebných python modulů stačí přejít do složky src a použít pip install -r requirements.txt Pro správnou funkci je třeba mít nainstalováno PostgreSQL sudo apt install postgresql a nástroj sudo apt install libpq-dev
